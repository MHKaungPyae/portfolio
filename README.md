# Portfolio — Min Htet Kaung Pyae

Personal portfolio website built with **React + Vite + Tailwind CSS**.

## 🚀 Live Site

[https://www.mhkaungpyae.dev](https://www.mhkaungpyae.dev/)

## 🛠 Tech Stack

- **React** — UI library
- **Vite** — Build tool & dev server
- **Tailwind CSS** — Utility-first styling
- **JavaScript (JSX)** — Component logic

## 📂 Project Structure

```
src/
├── assets/           # Images (profile photo, etc.)
├── App.jsx           # Main app — all sections in one component
├── main.jsx          # Entry point
└── index.css         # Tailwind imports & global styles
```

## ✨ Sections

| Section     | Content |
|-------------|---------|
| **Hero**    | Name, tagline, social links (GitHub, LinkedIn, Facebook) |
| **About**   | Bio — studying SE at Mae Fah Luang University, open to jobs |
| **Skills**  | Tech stack progress bars + certificates with credential links |
| **Projects**| TripNest, Match4U, Review Summarization, Sentiment Analysis |
| **Contact** | Email, GitHub, LinkedIn, Facebook links |
